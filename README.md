# Retirement Tax Planner 2026

A comprehensive retirement planning tool that helps you model and optimize your retirement income, tax liability, and withdrawal strategies for 2026.

## Features

- Model multiple income sources (work income, pensions, dividends, interest)
- Track retirement accounts (Traditional/Roth IRA, 401k)
- Track investments (stocks, crypto, precious metals)
- Calculate federal and state (Utah) tax liability
- Compare multiple withdrawal scenarios
- Fetch real-time stock prices
- Import/export your financial data

## Usage

Visit the live app: [Retirement Tax Planner](https://yateskd-lab.github.io/retirement-planner/)
